01
🎯
获客
Lead Generation
▾

通过多渠道开发境外客户，深耕拉丁美洲核心市场，做好市场调研与客户关系管理。

  • 线上电商平台推广（阿里巴巴国际站、Made-in-China、亚马逊等）
  • 线下参加海内外行业展会（广交会、电子展等），主动拓客
  • 转介绍依托现有客户口碑，获取优质买家推荐
  • 重点市场：巴西、智利、墨西哥，了解各国进口政策与认证要求
  • 巴西进口需注意 INMETRO 认证；墨西哥需关注 NOM 标准；智利需 SERNAC 法规
市场调研要点
分析目标市场消费者对电子产品的功能偏好（如充电速度、多语言界面）、价格区间、竞品情况，为后续报价与选型提供依据。
02
💹
报价
Price Quotation
▾

明确贸易条款与报价方式，结合市场调研确认产品电气参数，含装柜及仓库费用，注意含税/不含税口径。

  • FOB离岸价：货物到达指定港口后，费用与风险由买方承担
  • CFR成本+运费：我司负责运费，风险在装船后转移
  • CIF成本+保险+运费：我司承担运费与保险
  • 我司实际报价中，装柜费用与仓库费用均由我司负责
  • 报价前务必确认是否含增值税，区分含税价与不含税价
📋 电子产品认知 — 电气参数核对
依据目标市场电网特性，在报价阶段即确认以下核心电气参数：
市场电网电压电网频率插头类型认证要求
🇧🇷 巴西127V / 220V60HzNBR 14136（类似BR-2)INMETRO
🇲🇽 墨西哥127V60HzNMX/JEM 148NOM
🇨🇱 智利220V50HzCL type (L、N接地)SERNAC / SEC
⚡ 电气输入输出参数确认
输入参数：额定电压（V）/ 额定电流（A）/ 额定功率（W）/ 输入频率范围（Hz）
输出参数：输出电压（V）/ 输出电流（A）/ 输出功率（W）/ 波形与纹波
功能核验：是否支持宽电压输入（90–264V全球通用）/ 过压过流保护 / 待机功耗是否符合能效标准
📊 根据市场调研确定产品参数
结合市场调研结论：
· 巴西消费者偏好具备USB快充接口的电子产品（18W以上），需支持127V/220V双电压
· 墨西哥市场对Type-A/B接口兼容性需求高，NOM认证须包含能效标签
· 智利市场对功率因数（PF>0.9）和待机功耗（<0.5W）要求严格，优先推荐宽电压范围产品
依此确定产品规格书，与工厂确认能否达标后，再向客户正式报价
03
💳
支付方式确认
Payment Terms
▾

与客户协商并确认收款方式，兼顾安全性与便利性，根据客户资信评估结果确定支付条款。

  • 低风险信用证（L/C）：银行作为中间方，安全性高，适合新客户或大额订单
  • 常用电汇（T/T）：分前TT+后TT组合，灵活高效，适合老客户
  • 注意直接付款（D/P、D/A）：付款后交单，需评估客户资信风险
  • 选择支付方式前需完成客户资信审查
  • 对拉美新客户建议先采用 L/C 或 30% 前TT + 70% 见提单复印件付款
主流支付方式对比
L/C（信用证）风险最低，但手续繁琐、费用高；T/T（电汇）速度快，风险由合同比例决定；D/P（付款交单）有一定风险，慎用于新客。
04
📋
订货
Order Placement
▾

确认订单后签署采购合同，约定定金比例与账期，在ERP系统录入采购合同，产品规格须与技术确认单严格一致。

  • 通常由我司向工厂支付 定金 10%，余款按月结方式结算
  • 在 ERP 系统上准备并下达下单合同，确保信息与客户 PO 一致
  • 合同须明确：品名规格、数量、单价、交货期、包装要求
  • 订货前确认工厂备料周期，避免因原材料短缺延误交期
  • 大单建议加入质量索赔条款，保护己方权益
📋 订单产品技术确认
订货时须附《产品规格确认书》，由客户与工厂双方签字确认：
电学参数：输入电压/电流/功率 / 输出电压/电流/功率 / 频率 / 能效等级
功能要求：支持的电网制式 / 保护功能（过压/过流/短路）/ 接口类型（USB/Type-C/AC座）
性能指标：工作温度范围 / 防水防尘等级（IPX）/ 认证标识（INMETRO/NOM/SERNAC）
外观包装：颜色 / 材质 / 丝印语言（葡萄牙语/西班牙语）
📊 功能与性能确认（市场差异化）
· 巴西订单：确认产品宽电压支持（127V/220V），INMETRO认证标识必须丝印于机身，附葡萄牙语说明书
· 墨西哥订单：确认NOM能效标签（NOM-032）合规，电源线插头符合NMX标准，附西班牙语标签
· 智利订单：确认220V/50Hz专用规格，待机功耗<0.5W须符合智利能效标准，SEC认证编号须标注于外箱
技术确认书作为合同附件，具有同等法律效力
ERP录单流程
登录ERP → 新建采购订单 → 录入供应商、品号、数量、单价、交期 → 附技术确认书 → 审核提交 → 生成采购合同单据 → 发送工厂确认

备货与验货
📷 图片上传
▾

我司人员赴工厂验货，依照AQL抽样标准核对产品电气参数与功能是否符合技术确认书，提交验货图片与报告。

  • 与生产工厂和包装厂确认生产期限，安排验货日程
  • 我司验货人员提前到工厂，确认备货数量与包装状态
  • AQL按 AQL 抽样标准（通常 AQL 1.5/2.5）对电子产品进行抽样
  • 核验内容：外观、功能参数、标签合规性、包装完整性、接口适配性
  • 发现不合格品须立即通知工厂整改，重新验货后方可放行
  • 验货合格后填写验货报告，拍摄现场照片作为存档凭证
验货核检清单
① 产品外观无划痕/变形　② 功能参数符合规格书（电压/频率/功率）
③ 认证标志（CE/FCC/INMETRO/NOM）正确贴附且可读
④ 包装箱唛头（品名/数量/净重/毛重）及语言（葡语/西语）准确
⑤ 装箱数量与重量核对
⚡ 电气参数现场核验
使用万用表/示波器实测：
· 输入电压范围实测（是否覆盖目标市场电网：127V/220V 60Hz或220V 50Hz）
· 输出电压/电流稳定度测试
· 待机功耗实测（智利订单重点核查：<0.5W）
· 保护功能触发测试（过压/过流断电保护是否正常）

陆上运输与监装
📷 图片上传
▾

联系拖车提柜，我司人员赴仓库现场监装，核实装箱数量与柜号，拍摄封柜照片，记录铅封号。

  • 联系拖车公司，根据船期安排提柜时间（通常提前1-2天）
  • 我司监装人员到仓库/堆场，全程监督货物装柜
  • 核对每箱标签（唛头）、数量与装箱单一致
  • 记录集装箱号（Container No.）和铅封号（Seal No.）
  • 装柜结束后立即拍摄封柜照片，与铅封号一并留档
  • 联系报关公司提供装箱单、发票等单据，安排出口报关
  • 如涉及锂电池（危险品），须提前办理危险品证明文件
监装核验要点
① 货物件数与装箱单一致，无多装少装
② 外箱唛头（品名/目的港/客户名）与报关单一致
③ 集装箱箱况良好（无破损/漏水/异味）
④ 铅封（Seal）完好加锁后立即记录编号并拍照
⑤ 监装人员与司机确认交接，完成监装确认单
🚢 监装报告核心内容
日期时间 / 仓库地址 / 集装箱号 / 铅封号 / 货物件数
装箱单编号 / 监装人员签字 / 异常情况说明（如有）

报关与提单交付
Customs & B/L Delivery
▾

委托报关行完成出口清关，核对海运提单信息，确认通知人、收货人，正本提单交付客户。

  • 提前联系报关公司，提供装箱单、发票、合同、Form-A等单据
  • 确认货物 HS 编码，核实出口退税率
  • 出口报关时申报金额须与合同、发票保持一致，避免海关查验
  • 办理海运保险（All Risks），保额通常为货值的110%
  • 确认提单草稿（B/L Draft）无误后，通知货代出正本提单
  • 提单正本须在截止日期前寄送客户，避免延误清关
📋 提单（Bill of Lading）核对清单
提单字段核对要点
Shipper（托运人）我司公司名称、地址、联系电话，须与报关单一致
Consignee（收货人）客户在目的港的公司名称、地址；提单抬头须与客户指示一致（L/C条件下以开证行为收货人）
Notify Party（通知人）目的港货代或收货人，姓名+地址+电话须准确；到港后船公司凭此通知提货
Description（货物描述）品名规格英文描述，须与发票一致，禁止模糊表述
POL / POD装港（深圳/广州）与卸港（桑托斯/墨西哥城/瓦尔帕莱索）
Container No. / Seal No.须与监装记录一致，铅封号不得涂改
GW / NW毛重/净重，须与装箱单一致
Freight（运费）预付（PP）/到付（CC），须与贸易术语对应（FOB→CC，CIF→PP）
报关所需文件
商业发票（Commercial Invoice）/ 装箱单（Packing List）/
合同（Sales Contract）/ 提单草稿（B/L Draft）/
产地证（Form A / CO）/ 检验检疫证书（如需）

结汇
Settlement
▾

货款到账后办理外汇结汇手续，同步核销与申报出口退税，完成资金回笼。

  • 货款到达外汇账户后，通过银行办理结汇（USD → CNY）
  • 重要：结汇后及时进行外汇核销，更新出口核销档案
  • 退税根据货物 HS 编码与出口报关单，申报出口退税
  • 退税申报须在货物出口后180天内完成，确保单证齐全
  • 注意核销与退税申报的对应关系，防止单证不符导致退税被拒
  • 电子产品退税率一般为13%，具体以财税政策为准
退税申报所需文件
出口报关单（加盖核放章）/ 出口发票 / 装箱单 /
增值税发票 / 收汇水单 / 外汇核销单

风险防范
Risk Management
▾

全流程风险管控，通过资信审查与合同存档，有效规避交易风险与法律纠纷。

  • 核查资信审查：新客户合作前，通过第三方平台调查企业背景、信用记录、经营状况
  • 存档合同存单：所有合同、订单、往来邮件须完整存档（电子+纸质）
  • 购买出口信用保险（中信保），防范买方拒付/无力支付风险
  • 警惕网络诈骗，核实客户邮件域名与账户信息，防范商业欺诈
  • 遇纠纷优先协商，协商不成可申请贸仲委或ICC仲裁
  • 建议每单维护完整的"贸易档案"，包含沟通记录、单证及发货凭证
资信审查渠道
Dun & Bradstreet（邓百氏）/ 中国出口信用保险公司（中信保）/
商务部境外投资信息 / 行业协会推荐
常见风险类型
收汇风险（买方不付款）/ 质量纠纷（产品不符合技术确认书）/
政策风险（关税变化/贸易壁垒）/ 汇率风险（外汇波动）/ 认证风险（进口国拒绝清关）
